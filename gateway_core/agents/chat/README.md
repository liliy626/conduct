# gateway_core.agents.chat

## 1. 目录职责

- 普通聊天 Skill，服务非学校数据、非政策专用的轻量对话。

## 2. 输入

- 用户消息、模型客户端运行时上下文。

## 3. 输出

- 文本内容流或最终文本回答。

## 4. 核心文件

- `chat_skill.py`：普通聊天专家实现。

## 5. 数据流

- 入口层判定不进入 school/policy 专家时，调用聊天 Skill 生成回答。

## 6. 不负责什么（Boundary）

- 不查学校数据库。
- 不生成 EvidenceBoard 或多模态资产。

## 7. 修改这里时的注意事项

- 保持轻量；不要把学校业务规则放到这里。
